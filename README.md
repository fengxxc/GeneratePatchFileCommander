# CopyFileCommander

#### 前提

首先，要有python3的运行环境

 #### 配置文件config.json

`homePath`——字符串，项目所在的家目录

`workPaths`——数组[字符串]，你想要导出的文件们

`sourcePaths`——数组[字符串]，源码路径

`outputPath`——字符串，源码编译路径

`targetPath`——字符串，导出的文件存放的路径

`log`——布尔值，是否将日志保存成txt文件，是`true`，否`false`

#### 运行
直接运行main.py，或在当前目录的命令行里执行`python main.py`
